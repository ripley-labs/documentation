# NodeJS development

## CLI execution
```
# execute multiple commands in parallel
sudo npm install -g concurrently 

# pass environment variables as arguments to a command
sudo npm install -g cross-env

# pass environment variables from file to a command
sudo npm install -g dotenv-cli 

```

## Testing

```
# Unit test framework
sudo npm install -g jest

# static code analysis and fixer
sudo npm install -g eslint

# code style analysis and fixer
sudo npm install -g prettier

# run postman scripts in cli
sudo npm install -g newman
```
